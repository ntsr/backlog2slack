# backlog2slack
Post Backlog update message to Slack.

thx. https://qiita.com/kosuge/items/051922673cf57203f8db

## requirements
* Serverless Framework.

## deploy
1. Make `config.yml` from `config.yml.sample`.
2. Run following script.

```
$ npm run deploy
```

## remove
1. Run following script.

```
$ npm run remove
```
